# ionic v1 的tab app 模板
这里保存两个分支
- pre 是ionic原先的，未做任何修改。
- after 是模块化后的模板，结构如下：
```
├─ /www/ ·····················  项目www目录
└─┬─ areas/ ······················  所有模块所在目录 
  ├─── account/ ······················· 相关模块
  ├───── account.html ·····················模块下的html文件
  ├───── account_controller.js ············模块下的js文件
  ├───── account_route.js ·················模块下的js文件
  ├─── .../ ··························· 其他模块
  ├─ css/ ························  css 所在文件 
  ├─ img/ ························  img 所在文件
  ├─ js/ ·························  js 所在文件
  ├─── app.js ···························程序入口文件
  ├─── route.js ·························总路由文件
  ├─ lib/ ························  lib 所在文件
  └─ index.html ··················  入口文件
```
在这里调整结构，以便于日后的模块化开发，这里只做下演示
